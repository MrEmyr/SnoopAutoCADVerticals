# Resources

Application resources including icons and embedded files.

## Subfolders:

### **Icons/**
UI icons for:
- Tree view nodes
- Object types
- Commands
- Toolbar buttons

## Resource Files:
- Icons (.ico, .png)
- Images (.bmp, .png)
- Embedded resources

## Icon Requirements:
- 16x16 for tree nodes
- 32x32 for toolbar
- PNG with transparency preferred
